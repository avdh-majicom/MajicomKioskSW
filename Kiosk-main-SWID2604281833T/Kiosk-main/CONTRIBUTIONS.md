CONTRIBUTIONS.md

Purpose
- This file provides contribution guidance for humans.
- Keep this short; expand as needed.

Getting Started
- Use Arduino CLI to compile: arduino-cli compile --fqbn arduino:avr:mega Kiosk.ino
- Install required libraries if missing: U8g2, hd44780, Adafruit PN532.

Code Style
- Keep changes small and well-scoped.
- Prefer readable names and minimal side effects.

Testing
- Compile the sketch for the target board before submitting.

Submitting Changes
- Include a clear commit message.
- Summarize behavioral changes and any testing performed.
